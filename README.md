## git bash CLI

명령어
 - ./ : 현재 작업하고 있는 폴더
 - ../ : 현재 작업하고 있는 폴더의 부모 폴더
 - ~ : 홈 디렉토리
 - cd(change directory) : 경로 이동 방법
 - ls(list segment) : 현재 경로에 있는 디렉토리, 파일들을 표시
 - mkdir(make directory) : 새로운 디렉토리 생성
 - touch : 파일 생성
 - mv : 파일 이동
 - rm : 삭제

단축키
 + tab : 폴더/파일 이름 자동완성
 + ctrl + a : 커서가 맨 앞으로 이동
 + ctrl + e : 커서가 맨 뒤로 이동
 + ctrl + w : 커서가 앞 단어를 삭제
 + ctrl + l : 터미널 화면을 청소
 + ctrl + insert : 복사
 + shift + insert : 붙여넣기
